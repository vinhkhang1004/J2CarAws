### Mục tiêu tuần 3:
* Thực hành dịch vụ cơ sở dữ liệu NoSQL Amazon DynamoDB.
* Thiết lập hệ thống caching bộ nhớ với Amazon ElastiCache.
* Cấu hình phân phối nội dung toàn cầu bằng Amazon CloudFront.
* Tích hợp giám sát hoạt động hệ thống qua Amazon CloudWatch.
* **Lên văn phòng AWS tại TP.HCM (24/05/2026)**: Tham dự buổi đào tạo trực tiếp về VPC Security nâng cao (NACL, SG) và Multi-VPC.

### Kết quả đạt được tuần 3:
* Nắm vững cơ chế lưu trữ NoSQL DynamoDB với cấu trúc khóa chính Primary Key (Partition Key & Sort Key).
* Vận hành Redis Cluster qua ElastiCache giúp tối ưu tốc độ đọc dữ liệu.
* Tăng tốc tải trang web tĩnh nhờ phân phối qua các POP của CDN CloudFront.
* Thiết lập Dashboard giám sát và cảnh báo tự động trên CloudWatch.
* Hoàn thành buổi đào tạo tại văn phòng AWS, nắm vững cách hoạt động của Network Access Control List (NACL) ở mức subnet (khớp với Hình 1).
